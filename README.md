# Inter-Max website

Scop: proiectul pentru site-ul Inter‑Max (Construcții / Militar / Agro).

## Structură
- `docs/` — cerințe, copy, sitemap, wireframes
- `assets/` — logo, imagini, iconuri, exporturi
- `src/` — codul site-ului
- `notes/` — notițe, task-uri, backlog
